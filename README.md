# 川大课程表ScuTimetable
川大课程表安卓应用，模拟登录四川大学教务系统，获取课程表信息并展示，支持一键评教，课程表桌面插件。

软件已上架手机乐园，[下载地址](https://soft.shouji.com.cn/down/1555815.html)
